### 진행 방식

- 각 레포지토리룰 fork하여 자신의 브랜치에 구현한 다음, 자신의 브랜치로 PR한다.
- 각 미션 당 진행 기한은 각 미션 시작 시에 정한다.
- 미션 구현을 해오지 않은 경우, 벌금 10,000원을 청구한다.
- 정기 미팅 이외에도, 모르거나 막히는 부분이 있으면 단톡방에 적극적으로 물어본다.
